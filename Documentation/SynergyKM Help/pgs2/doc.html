SynergyKM Help

Synergy lets you easily share a single mouse and keyboard between multiple computers with different operating systems, each with its own display, without special hardware. It's intended for users with multiple computers on their desk since each system uses its own monitor(s).

Redirecting the mouse and keyboard is as simple as moving the mouse off the edge of your screen. Synergy also merges the clipboards of all the systems into one, allowing cut-and-paste between systems. Furthermore, it synchronizes screen savers so they all start and stop together.

SynergyKM is a mac os x preferences pane for the Synergy project.

SynergyKM additionally provides the ability to switch easily between multiple configurations (locations). Also limited Rendezvous/Bonjour support for Mac OS X only environment is included. SynergyKM automatically starts as soon as the user logs in.


System requirements

  • Two or more computers running Linux* (with X11/XTEST extension), Mac OS X (10.2 or higher) or Microsoft Windows 9x/NT/2000/XP.
  • A LAN (Home-/Office Network)

* Synergy has only been extensively tested on Linux and may not work completely or at all on other versions of Unix.


Getting started

First you have to decide which computer should be the synergy server, i.e. which keyboard and mouse you want to share with other synergy-enabled clients in your network.


Setup the server (Mac OS X)

  1. Open the SynergyKM preferences pane
  2. Optionally create a new location to preserve your current settings
  3. Select "Share my keyboard and mouse"
  4. Open the "Server Configuration" tab
  5. Add a screen with the name of your computer
  6. Add screens for your client computers with their respective names
  7. Click "Apply Now" when you are done
  8. If synergy is not yet running, start it using the "Turn Synergy On" button in the "General" tab

Setup a client (Mac OS X)

  1. Open the SynergyKM preferences pane
  2. Optionally create a new location to preserve your current settings
  3. Select "Connect to a shared keyboard and mouse"
  4. Open the "Client Configuration" tab
  5. Type the hostname or ip address of your server machine into the text field. If you want to use Rendezvous/Bonjour type in the actual screen name of the server instead of his address.
  6. Click "Apply Now" when you are done
  7. If synergy is not yet running, start it using the "Turn Synergy On" button in the "General" tab

Rendezvous/Bonjour support (Mac OS X)

Note: Rendezvous/Bonjour support is not yet completed. the functionality may change in future releases.

Using SynergyKM's Rendezvous/Bonjour function in a Mac OS X only environment features some advantages: different configurations/locations can be selected automatically and reconnects might be much faster.

In the server setups you don't have to make any modifications. In the client setup you have to type in the actual screen name of the server instead of it's address. the address is resolved automatically at runtime.
